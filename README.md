# **Social Network**

A single-page social network application connecting different users.

### Features
* Update users' personal profile and images.
* Send, receive, accept, delete friend requests.
* Live chat conversation among all users.
* Write on the friends' wall to leave message.
* List of friends, list of friend requests received, list of friend requests sent.

### Built with
React | Redux | Hooks | Node.js | Express.js | PostgreSQL | AWS | S3 | Socket.IO
